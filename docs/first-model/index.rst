My first model
==============

In this first tutorial, we are going to build SEAS model for
`normal and reverse <https://xkcd.com/1874/>`_ faulting on a planar fault.

We create the CAD model of the fault and generate the mesh in the :doc:`first step <mesh>`.
The material and friction parameters are set in a :doc:`Lua-script <lua-scripting>`.
Simulation parameters are set in the :doc:`parameter file <parameters>`.

.. tip::

   If you dislike copying text snippets, you can copy the complete tutorial files from the
   `example folder <https://github.com/TEAR-ERC/tandem/tree/master/examples/tandem/2d>`_.
